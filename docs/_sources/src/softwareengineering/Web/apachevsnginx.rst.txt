Apache VS. Nginx
=====================

* 参考：https://kinsta.com/jp/blog/nginx-vs-apache/

Webのプロトコルを簡単に言えば、DNSとIP要求を正しいコンピュータに転送する。
要求されたウェブページをホストしているコンピュータが要求に対して、ページを提供する。

Nginx, Apacheなどのソフトウェアはリクエストを処理して、結果を返す。


Apache
~~~~~~~

Apacheが最初にリリースされている。
Linuxの主要ディストリビューションにプリインストールされている。


Nginx
~~~~~~

「Nginxは、Apacheウェブサーバーのパフォーマンス制限に対処すべく生み出されました」

* 当初はApache インストール前のロードバランサーまたはリバースプロキシとして使用されていた
* そのあとはApacheからnginxへの動きが。
