今日の天気 (05-28 Tue)

5月28日の東京の天気予報ですが、時間帯ごとに変動があります。午前中は比較的晴れており、気温は21.3°Cから22.5°Cの間で推移します。しかし、午後から夜にかけては雨が降りやすくなり、特に夜には降雨量が増えることが予想されます。夕方から夜にかけては小雨やにわか雨があり、気温は22.2°Cから23.0°Cの間で推移します。 日中のUVインデックスは3.4と中程度であり、日照時間はゼロと予想されています。このため、直射日光による強い紫外線は少ないですが、外出時には念のため日傘を持って行くと安心です。特に晴れ間がある場合は、日差し対策が役立つでしょう。 全体としては、午前中と昼過ぎまでは比較的過ごしやすい天気ですが、午後から夜にかけての雨に注意が必要です。外出する際には雨具と日傘の両方を用意しておくと良いでしょう。
